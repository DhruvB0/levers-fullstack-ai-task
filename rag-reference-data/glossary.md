# Debt Collection Glossary

## Account Statuses

| Status | Meaning |
|--------|---------|
| ACTIVE | In active collection. Contact and negotiation permitted. |
| DISPUTED | Consumer has disputed the debt. Collection paused until verified. |
| CEASE_DESIST | Consumer requested no further contact. Only legal notices permitted. |
| ATTORNEY_REP | Consumer has an attorney. All communication goes through the attorney. |
| BANKRUPTCY | Bankruptcy filed. Automatic stay — ALL collection prohibited. |
| SCRA_HOLD | Active-duty military. SCRA protections apply. Interest capped at 6%. |
| SETTLED | Resolved via settlement. No further collection. |
| PAYMENT_PLAN | Consumer on a structured repayment schedule. |

## Key Regulatory Terms

| Term | Definition |
|------|------------|
| FDCPA | Fair Debt Collection Practices Act — federal law governing third-party debt collection. |
| Regulation F | CFPB rule implementing the FDCPA (effective Nov 2021). Covers validation notices, call limits, and digital communications. |
| Mini-Miranda | Required disclosure on every call: "This is an attempt to collect a debt..." |
| Validation Notice | Written notice with debt details and consumer rights, required within 5 days of first contact. |
| Statute of Limitations | Time window to file a lawsuit. Varies by state (typically 3–10 years). Expiration doesn't erase the debt. |
| Automatic Stay | Bankruptcy protection halting all collection immediately upon filing. |
| Deficiency Balance | Amount owed after collateral (car, house) is sold. |
