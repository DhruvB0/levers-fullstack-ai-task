# Call Scripts — Debt Collection

## Opening Script (Mandatory)

"Hello, my name is [AGENT NAME] and I'm calling from [COMPANY NAME]. This is an attempt to collect a debt. Any information obtained will be used for that purpose. Am I speaking with [CONSUMER NAME]?"

## If Consumer Disputes the Debt

"I understand. You have the right to dispute this debt in writing within 30 days of receiving our validation notice. If you do, we will pause collection and send you verification. Would you like our mailing address?"

## If Consumer Requests No More Calls

"I understand. Please send a written request to [COMPANY ADDRESS] and we will honor it. Please note this does not eliminate the debt. We may still notify you of any legal actions."

## Payment Negotiation

"The current balance is [AMOUNT]. We can offer:
1. A lump-sum settlement of [SETTLEMENT AMOUNT] ([X]% of balance).
2. A payment plan of [MONTHLY AMOUNT] over [X] months.
3. Tell me what works for you and we'll see what we can arrange."

## Voicemail (Limited-Content Message)

"Hello, this is [AGENT NAME] from [COMPANY NAME] calling for [CONSUMER NAME]. Please return my call at [PHONE NUMBER]. Reference number [ACCOUNT NUMBER]. Thank you."

Note: This voicemail intentionally does not mention the debt — it qualifies as a limited-content message under Regulation F.
